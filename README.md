# 🍳 Кухарня

Три повара. Один готовит каждый день. Остальные голосуют — засчитать или нет.

Телеграм мини-приложение для отслеживания домашней готовки с голосованием и статистикой.

-----

## Как это работает

Каждый день один из трёх поваров готовит еду, фотографирует блюдо прямо в приложении и отправляет на голосование. Двое других смотрят фото и голосуют — одобрить или отклонить. Набрал 2 голоса «за» — день засчитан. Система считает стрики и следит за балансом: никто не должен готовить сильно чаще других.

```
Повар готовит → фото → голосование → ✅ засчитано / 🚫 отклонено
```

## Структура проекта
 у кого план на будущее, какие дни вошли в стрик 🔥

**Голосование** — пост уходит в Telegram всем поварам, голосуют прямо там инлайн-кнопками. Результат виден и в приложении и в боте.

**Планирование** — можно назначить повара на день вперёд (до 7 дней)

**Стрик** — считается серия дней подряд где кто-то готовил. Сбрасывается если баланс готовки между поварами нарушился (разница больше 1)

**Статистика** — сколько раз каждый готовил, постил, голосовал

**Админка** — отдельная панель для управления постами, планированием и пользователями
