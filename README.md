# 🍳 Кухарня

Три повара. Один готовит каждый день. Остальные голосуют — засчитать или нет.

Телеграм мини-приложение для отслеживания домашней готовки с голосованием и статистикой.

---

## Как это работает

Каждый день один из трёх поваров готовит еду, фотографирует блюдо прямо в приложении и отправляет на голосование. Двое других смотрят фото и голосуют — одобрить или отклонить. Набрал 2 голоса «за» — день засчитан. Система считает стрики и следит за балансом: никто не должен готовить сильно чаще других.

```
Повар готовит → фото → голосование → ✅ засчитано / 🚫 отклонено
```

---

## Фичи

**Календарь** — видно кто готовил каждый день, у кого план на будущее, какие дни вошли в стрик 🔥

**Голосование** — пост уходит в Telegram всем поварам, голосуют прямо там инлайн-кнопками. Результат виден и в приложении и в боте.

**Планирование** — можно назначить повара на день вперёд (до 7 дней)

**Стрик** — считается серия дней подряд где кто-то готовил. Сбрасывается если баланс готовки между поварами нарушился (разница больше 1)

**Статистика** — сколько раз каждый готовил, постил, голосовал

**Админка** — отдельная панель для управления постами, планированием и пользователями

## Стек

| Часть | Технология |
|-------|-----------|
| Фронтенд | Vanilla JS, Firebase SDK |
| Бэкенд | Vercel Serverless Functions (Node.js) |
| База данных | Firebase Realtime Database |
| Авторизация | Google OAuth (веб) + Telegram initData (бот) |
| Изображения | Cloudinary |
| Деплой | Vercel (main → прод, dev → стенд) |
| Бот | Telegram Bot API |

## Структура проекта

```
/
├── index.html          # Главная страница (мини-апп)
├── admin.html          # Админ-панель
├── config.js           # Фронтенд-конфиг (читает из window.ENV)
├── app.js              # Инициализация Firebase, глобальное состояние
├── calendar.js         # Рендер календаря
├── dayView.js          # Панель дня (действия, планирование)
├── posts.js            # Рендер постов, голосование
├── camera.js           # Камера, захват фото, загрузка
├── stats.js            # Стрики, лидерборд, статистика
├── style.css           # Стили (light + dark theme)
│
├── api/
│   ├── env.js          # Отдаёт публичный конфиг фронтенду
│   ├── bot.js          # Telegram webhook
│   ├── upload.js       # Загрузка фото → Cloudinary → Firebase
│   ├── notify.js       # Cron: напоминание повару
│   ├── setwebhook.js   # Регистрация Telegram webhook
│   └── admin/
│       ├── auth.js     # Логин в админку (bcrypt + JWT)
│       ├── posts.js    # CRUD постов
│       ├── planning.js # CRUD планирования
│       └── users.js    # Управление пользователями
│
├── lib/
│   ├── firebase.js     # Firebase Admin SDK
│   ├── config.js       # Серверные константы из env
│   ├── adminAuth.js    # Middleware проверки JWT
│   └── telegram.js     # Хелперы Telegram Bot API
│
├── messages.js         # Тексты сообщений бота
├── .env.example        # Шаблон переменных окружения
└── vercel.json         # Роутинг Vercel
```
